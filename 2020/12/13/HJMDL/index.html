皇家马德里

1.初步

拿到压缩包后解压。得到两个文件，falg.zip和key.zip，尝试打开两个文件但是发现都加密了，根据文件名字提示，我就先从key.zip入手。遇到加密的压缩包应该先尝试一下是否是伪加密，用Winhex打开，搜索十六进制数值504B0102
,对其后面的第九第十位改为0

###即修改后可以直接打开，打开后得到falg.jpg，发现打不开，用Winhex打开得到

BBAA AA ABBA BABA ABA BABB ABBA B BBB

2.第二步

根据密码表现,猜测是培根密码或者摩斯电吗加密，尝试培根无果，于是尝试莫斯电码，先把A变成-B变成. 结果解出来的东西并不像一个正规的东西，于是反着来把A变成点B变成杠，得到–.. .. .–. -.-. .-. -.– .–. - —
解出ZIPCRYPTO. 而ZIPCRYPTO就是falg.zip的密码，成功解压falg.zip.解出两个文件flag.png 和密码.jpg。

打开密码.jpg图片看到上面有一个提示lsb隐写，但是一般lsb隐写多用于bmp文件和png文件。所以先尝试别的方法。根据题目本来给的提示：好像用到了好几次base，用Winhex打开密码.jpg，由于FFD9是jpg文件的文件尾，那么后面的内容就应该是一些需要解密的东西了,结合题目提示，将文件尾部之后的内容十六进制翻译得到：5’:{hAY:[UvvR base家族！！！！！1把无关内容删掉得5’:{hAY:[UvvR
由于提示要用几次base,先base92得到6281M:Z;wS
再拿去经过base91解密得到真正隐藏的字符lsbjiami
。

3.第三步

那下一步就要着手flag.png了，尝试了各种方法依旧无果，卡了一天，于是决定去看看别的png隐写题是怎么解的，终于找到一个之前从来没用过的解密方式，具体参考https://blog.csdn.net/fuzz_nancheng/article/details/53384353

于是便着手搭建环境，此处需要脚本，脚本地址https://github.com/livz/cloacked-pixel

此处我参考的上面那篇博客，也用的kali，且python版本2.7，由于前一晚上电脑出问题了导致环境都崩了，于是得重新弄，想要用到脚本，必须得用到PIL库,prycryto，pycrytodome

而大家百度到的什么pip install crypto就不要用了，安装的东西都是残缺的，直接pip install xxxxx，如果没有换源的话可以百度一下pip换源，于是变成这样 pip install xxxxx -i 源的地址，这样下载速度会很快，否则可能会报times out的错误，都安装好了之后就可以使用脚本了，将上面链接下载的东西放入kali，解压，在大文件夹下进行基本操作: python2 lsb.py extract flag.png <解密下的内容放的文件> lsbjiami（这是密码）

如python2 lsb.py extract flag.png Sapphire.txt lsbjiami
，运行成功后会出现一个文件，打开发现第一行写的是“flag不在这”，我们直接拉到底就能看到flag了，完结撒花。

PS:如若运行脚本出错，把错误提示拿去百度翻译，看看自己少了什么内容，继续百度找方法安装即可。
